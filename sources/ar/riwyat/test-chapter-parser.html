// New: DOM-level watermark removal before text extraction
fun removeWatermarks(doc: Document): Document {
    doc.select("span[aria-hidden=true][role=presentation]").remove()
    doc.select("span[data-nosnippet=true]").remove()
    return doc
}

How the watermark works on cenele.com

Hidden watermark HTML structure

<p>Actual chapter text here.<span
    aria-hidden="true"
    role="presentation"
    data-nosnippet="true"
    translate="no"
    data-bkq0thcb="1"
    style="position:absolute;left:0;top:0;width:1px;height:1px;overflow:hidden;opacity:0;pointer-events:none;filter:blur(0.5px);font-size:16px;line-height:1.6;">
  هذا نص تمويهي من موقع فضاء الروايات فقط...
  https://cenele.com/ #BkQ0thCB
</span></p>

The watermark text is invisible (CSS hidden) but Jsoup's .text() still extracts it.
Our fix: doc.select("span[aria-hidden=true][role=presentation]").remove() deletes these spans from the DOM before calling .text().
